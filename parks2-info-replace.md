# parks2-info-replace (Tampermonkey Script)

这是一个用于 **NAMCO Parks (ナムコパークス)** 个人中心页面的信息替换脚本。它可以帮助用户在页面上临时替换显示的个人信息（如姓名、地址、性别等），并支持从当前页面一键提取真实信息作为模板。

## 🚀 主要功能

- **精确替换**：针对 `member_mypage.html` 页面结构，根据标签（如“氏名（漢字）”）精确替换对应的数值。
- **自定义配置**：提供图形化设置界面，支持配置姓名、生年月日、性别、邮编、地址、电话等。
- **性别下拉选择**：性别字段支持从预设选项（男性、女性、あてはまらない、回答しない/非表示）中快速选择。
- **灵活替换**：设置项放空时将不进行替换，保持页面原始数据。
- **数据提取**：支持一键“提取当前页面数据”，自动获取页面原始真实信息并存入设置，方便作为模板使用。
- **原始值保护**：内置原始值快照逻辑，即使在已替换信息的页面上点击提取，也能准确找回账号的真实原始数据。
- **本地保存**：所有配置均保存在浏览器的 `localStorage` 中，刷新页面或下次访问时自动应用。
- **开源协议**：采用 MIT 协议发布。

## 🛠 使用说明

1. **安装脚本**：将 `parks2-info-replace.js` 的内容复制到 Tampermonkey (油猴) 插件中并保存。
2. **生效页面**：脚本仅在 `https://parks2.bandainamco-am.co.jp/member_mypage.html` 生效。
3. **打开设置**：
   - 点击浏览器右上角的 Tampermonkey 图标。
   - 在菜单中选择 `🔧 打开替换设置`。
4. **配置信息**：
   - 在弹窗中输入想要显示的替换值。
   - 点击 `保存并应用` 立即生效。
5. **提取原始数据**：
   - 如果您想快速抓取当前账号的真实信息，点击 `提取当前页面数据`。
   - 系统会弹出确认框，确认后会将页面真实数据填入设置框（已考虑防替换 bug，能准确提取原始值）。

## 📝 字段列表 (支持替换)

- 氏名（漢字）
- 氏名（カナ）
- 生年月日
- 性別 (提供下拉菜单)
- 郵便番号
- 都道府県
- 市区町村
- 丁目・番地
- 電話番号

> **注意**：`ニックネーム` (昵称) 和 `メールアドレス` (邮箱) 已被排除在替换范围之外，将始终显示原始数据。

## ⚙️ 技术细节

- **URL 匹配**：`@match https://parks2.bandainamco-am.co.jp/member_mypage.html`
- **数据存储**：使用 `localStorage` 键名 `personal_info_replacements`。
- **快照机制**：在首次处理页面元素时，通过 `data-original-value` 属性锁定原始数据，防止提取到已替换的虚假信息。
- **动态监听**：使用 `MutationObserver` 监听 DOM 变化，确保异步加载的内容也能被处理。
